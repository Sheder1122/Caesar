# Шифр Цезаря — криптоанализатор

Консольное приложение для шифрования и дешифрования текстовых файлов с использованием шифра Цезаря. Поддерживает расшифровку методом brute force и статистическим анализом на основе эталонного текста.

## Основные возможности

1. **Шифрование файла** – сдвиг каждого символа алфавита на заданный ключ (сдвиг).
2. **Дешифрование по ключу** – обратное преобразование при известном сдвиге.
3. **Brute force** – автоматический перебор всех возможных ключей и выбор наиболее правдоподобного результата на основе частотного анализа букв.
4. **Статистический анализ** – определение сдвига путём сравнения частот букв в зашифрованном тексте с эталонным текстом.
